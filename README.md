# IBMLuxSensor

용량이 커서 안올라가 또 다시 구글 드라이브 링크로 첨부합니다

중간에 메타데이터 수정하는 부분은 바로 리부트를 눌렀더니 고쳐지지 않아 2번 정도 더 누르고 변경이 되었습니다

이 과정에서 1분정도 시간이 지체되어 편집하였는데 혹시 몰라 원본 영상도 같이 첨부합니다. 감사합니다
